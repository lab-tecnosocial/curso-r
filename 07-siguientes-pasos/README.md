# Siguientes pasos

¡Llegaste al final del curso! Ahora conoces los fundamentos de R y el
tidyverse. Esta lección te orienta hacia dónde ir desde aquí.

## ¿Qué aprendiste?

A lo largo del curso cubriste:

-   R como intérprete: tipos de datos, variables, funciones
-   Estructuras de datos: vectores, matrices, factores, data frames,
    listas
-   Estructuras de control: condicionales, bucles, funciones `apply`
-   Tidyverse: `dplyr`, `ggplot2`, `tidyr`, `stringr`, `lubridate`,
    `readr`
-   Limpieza de datos con `janitor`
-   Análisis básico de texto con `tidytext`

## Recursos gratuitos para continuar

### Libros online (gratis)

-   **[R for Data Science, 2ª edición](https://r4ds.hadley.nz/)**
    (Hadley Wickham, 2023) — El libro de referencia del tidyverse. En
    inglés, pero muy visual y práctico.
-   **[R for Data Science en español](https://es.r4ds.hadley.nz/)** —
    Traducción de la 1ª edición al español.
-   **[Text Mining with R](https://www.tidytextmining.com/)** — Para
    profundizar en análisis de texto con tidytext.

### Práctica interactiva

-   **[swirl](https://swirlstats.com/)**: aprende R directamente en la
    consola de R con ejercicios interactivos. Instálalo con
    `install.packages("swirl")`.
-   **[Posit Cloud](https://posit.cloud/)**: Posit RStudio en el
    navegador, sin instalar nada. Muy útil para compartir proyectos o
    trabajar desde cualquier computadora.

### Datos para practicar

-   **[TidyTuesday](https://github.com/rfordatascience/tidytuesday)**:
    cada semana se publica un dataset nuevo para practicar visualización
    y análisis. La comunidad comparte sus resultados en redes sociales.
-   **[Instituto Nacional de Estadística
    (Bolivia)](https://www.ine.gob.bo/)**: datos oficiales sobre
    Bolivia.
-   **[CEPAL](https://estadisticas.cepal.org/)**: datos
    socioeconómicos de América Latina.
-   **[Kaggle](https://www.kaggle.com/datasets)**: miles de datasets
    públicos sobre cualquier tema.

## Comunidades

-   **[LatinR](https://latin-r.com/)**: conferencia latinoamericana
    sobre R para investigación y datos. Tiene ponencias en español y
    portugués.
-   **[R-Ladies](https://rladies.org/)**: red global para promover la
    diversidad en la comunidad R. Hay capítulos en Bolivia y otros
    países latinoamericanos.
-   **[Foro de Posit Community](https://community.rstudio.com/)**: el
    lugar donde hacer preguntas sobre R, dplyr, ggplot y todo el
    ecosistema.

## ¿Qué aprender después?

Dependiendo de tus objetivos:

| Objetivo | Qué aprender |
|---|---|
| Reportes reproducibles | R Markdown avanzado o Quarto |
| Estadística inferencial | `infer`, modelos lineales con `lm()` |
| Ciencia de datos / machine learning | Paquete `tidymodels` |
| Aplicaciones web interactivas | `shiny` |
| Datos espaciales / mapas | `sf`, `ggmap`, `leaflet` |
| Big data | `arrow`, `duckdb` |
| Automatización | Scripts programados, `targets` |

## Un último consejo

La mejor forma de aprender R es usarlo con datos que te importen. Busca
un problema real de tu trabajo o área de interés, importa los datos en R
y trata de responder una pregunta concreta. Vas a equivocarte, buscarás
en Google, leerás documentación — y así es exactamente como se aprende.
